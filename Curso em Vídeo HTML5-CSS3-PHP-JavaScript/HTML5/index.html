Dicionário de tags HTML5

O que faz cada tag

Vamos começar pela tag da primeira linha:
<!DOCTYPE html>: Define o tipo do documento como HTML;

Mais adiante, temos a tag <html>: Esta tag indica o bloco onde o seu código html deverá ser escrito. Nesta tag podemos utilizar o parâmetro <html lang="pt-br">, por exemplo. assim, podemos mudar o idioma
deste documento. Mas este tag não traduz!

A próxima tag seria <hea>: Esta tag é responsável por toda a parte de configurações da página.
Ela que é responsável por: definir a codificação, definir o título da página, definir o link con a folha de estilos CSS3, etc.
Lembre-se de que esta tag tem fechamento: </head>

Agora passaremos para a parte de dentro da tag <head></head>:
Dentro da tag <head></head>, a primeira tag que convém que seja escrita é a tag <meta> que não tem fechamento.
Essa tag será escrita no mínimo duas vezes mas, com adições diferentes. Veja adiante:
<meta charset="UTF-8"> ---- Define a codificação da página para que possamos utilizar acentos e outras coisas; <meta name="viewport" content="width=device-width, initial-scale=1.0> ---- Configura a página para ocupar todo o espaço oferecido pelo navegador, podendo assim, se ajustar de acordo com o tamanho da tela do navegador.
